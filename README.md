# 약로그(DrugLog) 기반 약물 정보/상호작용 분석 프로젝트

## 1. 프로젝트 소개
본 프로젝트는 공개된 DrugLog 데이터를 활용하여 약물 관련 정보를 탐색하고, 약물 간 상호작용 및 사용자 관점에서 유의미한 패턴을 확인할 수 있도록 구성한 프로젝트입니다.  
데이터 분석 결과를 보다 직관적으로 확인할 수 있도록 Streamlit 기반 대시보드 형태로 구현했습니다.

## 2. 프로젝트 목적
- 공개 약물 데이터를 활용한 데이터 분석 및 서비스 구현 경험 축적
- 약물 정보 및 상호작용 데이터를 사용자 친화적으로 시각화
- 데이터 기반 문제 정의부터 분석, 구현, 배포까지의 전 과정 경험

## 3. 주요 기능
- 약물 정보 조회
- 약물 상호작용 관련 데이터 탐색
- 조건별 필터링 및 검색
- Streamlit 기반 시각화 대시보드 제공

## 4. 사용 데이터
본 프로젝트는 공개된 DrugLog 데이터를 활용하였습니다.  
데이터는 공개 포털에서 제공되는 오픈 데이터를 기반으로 하며, 저장소에는 용량 및 라이선스 이슈를 고려하여 원본 데이터를 포함하지 않았습니다.

- 데이터 출처: [데이터 출처 링크 삽입]
- 데이터 형태: CSV / JSON / 기타
- 활용 목적: 비상업적 프로젝트 및 학습/포트폴리오 목적

## 5. 데이터 처리
프로젝트 진행 과정에서 다음과 같은 전처리 및 분석 과정을 수행했습니다.

- 결측치 및 중복 데이터 처리
- 필요한 컬럼 선택 및 구조화
- 상호작용 관련 정보 정리
- 사용자 탐색에 적합한 형태로 가공

## 6. 기술 스택

- Streamlit
- GitHub
